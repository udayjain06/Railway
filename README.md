> Built for **FAR AWAY 2026** — India's Biggest International Hackathon | Railways Theme
---

## 🎯 Problem Statement

Indian Railways carries **23 million passengers daily** across 68,000 km of track. Yet:
- **55%+ of accidents** are caused by human error (loco pilot fatigue, signal failures)
- **₹1,700 crore** lost annually due to train delays and cascading disruptions
- **Zero real-time** fatigue monitoring exists for loco pilots
- Maintenance is **reactive, not predictive** — faults found after failure

**RailSaarthi** is a unified AI operations platform that tackles all of these simultaneously.

---

## 🧩 Modules

### 🔴 Fatigue Guard
Real-time loco pilot fatigue detection using **MediaPipe Face Mesh** running directly in the browser.
- PERCLOS (eye closure %) calculated across a 10-second rolling window
- Head nod detection for microsleep alerts
- Real-time PERCLOS line chart (last 60 seconds)
- **AI Incident Report** generated by Claude when danger threshold is crossed
- Score resets instantly when face leaves frame — no false positives on reload

### 🟡 Delay Radar
Live train delay tracking with cascade prediction.
- Integrates with **indianrailapi.com** for real train data
- Cascade model predicts downstream delays (if Train A is late, Train B gets delayed too)
- Interactive **Leaflet.js map** with color-coded train positions
- Auto-falls back to realistic simulation if API unavailable

### 🟢 Platform Pulse
Coach-level occupancy heatmap for passenger guidance.
- Simulates real-time coach load across SL, 3A, 2A, 1A, GEN coaches
- Recommends which coach to board for maximum space
- Updates every 10 seconds

### 🔧 Predictive Maintenance AI
Fleet-wide sensor monitoring with failure prediction.
- Monitors 8 sensor types: Wheel Bearing Temp, Axle Vibration, Brake Pad Thickness, Pantograph Pressure, Engine Oil Pressure, Suspension Deflection, Wheel Flange Wear, AC Compressor Load
- Severity classification: Low → Medium → High → Critical
- Predicted failure timeline per component
- **Claude AI fleet analysis** — one-click AI assessment of entire fleet health

### 🚨 Incident Command Center
Unified emergency response board that ties all modules together.
- Auto-generates incident tickets from Delay Radar & Maintenance alerts
- Full incident lifecycle: Open → Assigned → In Progress → Resolved
- Team assignment, timeline tracking, resolution time analytics
- **Claude AI summary** per incident for control room operators
- Manual incident creation for field-reported issues

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- Python 3.9+
- Git

### 1. Clone the repo
```bash
git clone https://github.com/udayjain06/Railway.git
cd railsaarthi
```

### 2. Start Backend
```bash
cd backend
pip install fastapi uvicorn numpy pillow
uvicorn main:app --reload --port 8000
```
Backend runs at `http://localhost:8000`

### 3. Start Frontend
```bash
cd frontend
npm install
npm run dev
```
Frontend runs at `http://localhost:3000`

### 4. Open in browser
```
http://localhost:3000
```
